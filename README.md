```
## 15.3 Key Metrics That Actually Matter:

> **Not all metrics are equal.** Here's what to focus on at each stage:

```
MONTH 1-2 (LAUNCH PHASE):
Focus on: Reel Views + Follower Growth Rate + Profile Visits
Why: We need REACH first. Are people discovering the content?
Ignore: Revenue (too early to judge)

MONTH 3-4 (GROWTH PHASE):
Focus on: Saves + Shares + DMs + Link Clicks
